# ASPIS — Roadmap

The public, canonical roadmap lives at the repository root:
**[`/ROADMAP.md`](../../ROADMAP.md)**.

It covers the **six parts** — install & onboarding · the production system ·
tracing · intelligence · surfaces (dashboard) · scale — with the problem each
solves, what is shipped (Parts 1–2 today), what is next (Part 3, tracing), and
what ASPIS deliberately does **not** do yet.

Per-feature delivery history is captured in
[`RECENT_CHANGES.md`](RECENT_CHANGES.md); the dated design decisions behind each
part are in [`DECISIONS.md`](DECISIONS.md).
