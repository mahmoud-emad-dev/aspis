# Attributions

ASPIS is not a clone of any single tool — it harvests good ideas from prior art and
re-expresses them in its own file-first, deterministic design. This file records the work
ASPIS has adapted, with thanks.

## GitHub Spec Kit — MIT License

ASPIS's planning discipline — the **spec → plan → tasks → implement** sequence with
prerequisite gating — and its `plan-critic` skill (the cross-artifact "analyze" consistency
check) are a lean adaptation of ideas from
[GitHub Spec Kit](https://github.com/github/spec-kit). This lineage is acknowledged in
`.aspis/context/IDENTITY.md` and `.aspis/context/CORE_LOOP.md`. Spec Kit is distributed
under the MIT License.
